## 💻 Installation

Clone the repository

```bash
git clone https://github.com/ShineGood/portfolio.git
```

Navigate into the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

The application will be available at

```
http://localhost:5173
```
